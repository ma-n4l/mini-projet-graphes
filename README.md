# Mini-Projet N°5 — Détection de Communautés Épidémiques

## Description
Modélisation des réseaux d'interactions physiques entre individus
et application de l'algorithme LPA pour détecter des communautés
de propagation d'infection.

## Dataset
- **Nom :** Contact High School
- **Source :** http://networkrepository.com
- **Noeuds :** 327 individus
- **Relations :** 5 818 contacts physiques

## Technologies
- Neo4j Desktop 2.1.4
- Python 3.12
- NetworkX
- Gephi

## Structure du projet
- `data/` : fichiers CSV du dataset
- `partie1_neo4j/` : scripts Python + requêtes Cypher
- `partie2_networkx/` : modélisation NetworkX
- `partie3_lpa/` : algorithme LPA
- `rapport/` : rapport LaTeX
